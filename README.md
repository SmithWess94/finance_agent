# 🍵 Финансовый агент "Наставник"

Персональный ИИ-финконсультант для владельца чайного магазина.
Работает через Telegram, ведёт учёт в Google Sheets, обучен на 5 классических книгах о деньгах.

---

## 🎯 Цель проекта

Помочь пользователю:
1. **Погасить долги перед поставщиками чая**
2. **Накопить финансовую подушку** (рекомендую 3-6 месяцев расходов)
3. **Выстроить дисциплину** учёта доходов и расходов
4. **Изменить мышление** через ежедневное применение принципов из книг

---

## 👤 Профиль пользователя

- **Бизнес:** чайный магазин, проведение чайных церемоний и саунд-хилингов
- **Род деятельности:** чайный мастер и проводник церемоний
- **Виды услуг:**
  - Чайные церемонии в собственной чайной
  - Выездные общие чайные церемонии
  - Заказные чайные церемонии
  - Саунд-хилинги
- **Тип дохода:** нестабильный (сезонность, выручка зависит от дня)
- **Главные долги:** перед поставщиками чая
- **Горизонт цели:** 1-3 года

## 🏗 Архитектура

```
Пользователь (Telegram)
        ↓
Telegram Bot (python-telegram-bot)
        ↓
Claude API (Anthropic) ← Knowledge Base (5 книг)
        ↓
Google Sheets API (учёт доходов/расходов/долгов)
        ↓
APScheduler (ежедневные напоминания 3 раза в день)
```

| Компонент | Технология |
|-----------|------------|
| Язык | Python 3.11+ |
| LLM | Anthropic Claude API (claude-opus-4-7) |
| Мессенджер | python-telegram-bot |
| Хранилище | Google Sheets (через gspread) |
| Планировщик | APScheduler |
| Конфиг | .env + python-dotenv |

## 📅 Расписание взаимодействия

| Время | Событие |
|-------|---------|
| **08:00** | Утро: мотивация + вопрос «Как сегодня заработаешь?» |
| **14:00** | День: проверка прогресса, напоминание о фиксации операций |
| **21:00** | Вечер: итоги дня, ввод доходов/расходов, рефлексия |
| **21:30** | Проверка: если за день ничего не записано → напоминание |

## 💬 Команды бота

| Команда | Что делает |
|---------|------------|
| `/start` | Приветствие, знакомство, настройка |
| `/доход <сумма> <описание>` | Добавить доход |
| `/расход <сумма> <категория>` | Добавить расход |
| `/долг <поставщик> <сумма>` | Добавить/обновить долг |
| `/погасил <поставщик> <сумма>` | Частично погасить долг |
| `/итоги` | Отчёт за день/неделю/месяц |
| `/совет` | Получить совет на основе книг |
| `/цель` | Посмотреть прогресс по цели |

Естественный язык тоже работает: «заработал сегодня 5000 с продажи пуэра» → бот сам распарсит.

## 📚 База знаний (5 книг)

Положи в папку `knowledge_base/` текстовые выжимки/конспекты книг:

1. **`kiyosaki.md`** — Богатый папа, бедный папа (денежный поток, актив vs пассив)
2. **`babylon.md`** — Самый богатый человек в Вавилоне (7 правил, 10% отложи себе)
3. **`housel.md`** — Психология денег (поведение > знания, сложный процент)
4. **`hill.md`** — Думай и богатей (мышление, цели, постоянство)
5. **`schaefer.md`** — Путь к финансовой свободе (план на 7 лет, 4 ступени богатства)

Эти файлы агент загружает в контекст системного промта при каждом запросе.

## 📊 Структура Google Sheets

Создай таблицу «Финансы Чайного Магазина» с 4 листами:

### Лист 1: «Доходы»
| Дата | Сумма | Источник | Категория | Комментарий |

### Лист 2: «Расходы»
| Дата | Сумма | Категория | Комментарий |

Категории расходов: `Закупка чая`, `Аренда`, `Реклама`, `Бытовое`, `Личное`, `Другое`

### Лист 3: «Долги поставщикам»
| Поставщик | Изначальный долг | Погашено | Остаток | Срок | Статус |

### Лист 4: «Цели и прогресс»
| Дата | Подушка (₽) | Общий долг (₽) | Комментарий |

## 🚀 План запуска (пошагово)

### Шаг 1. Подготовка окружения
```bash
cd ~/Documents
git init finance_agent
cd finance_agent
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### Шаг 2. Получить API ключи
1. **Anthropic API**: https://console.anthropic.com/ → API Keys
2. **Telegram Bot**: напиши @BotFather в Telegram → `/newbot` → получи токен
3. **Google Sheets API**:
   - https://console.cloud.google.com/ → создай проект
   - Включи Google Sheets API и Google Drive API
   - Создай Service Account → скачай JSON
   - Открой свою таблицу → поделись с email сервисного аккаунта

### Шаг 3. Заполни `.env`
```
ANTHROPIC_API_KEY=sk-ant-...
TELEGRAM_BOT_TOKEN=...
TELEGRAM_USER_ID=твой_telegram_id
GOOGLE_SHEET_ID=ID_из_ссылки_на_таблицу
GOOGLE_CREDENTIALS_PATH=config/google_credentials.json
```

### Шаг 4. Запуск
```bash
python src/bot.py
```

Напиши боту в Telegram `/start` — он поздоровается.

## 🧠 Как работает связка с Claude Code

Ты открываешь проект в VS Code, запускаешь Claude Code, и он видит всю структуру.
Можешь давать ему команды типа:
- «Добавь команду /отчёт_неделя»
- «Улучши промт наставника, сделай его жёстче»
- «Добавь график расходов в отчёт»
- «Сделай кнопочное меню в Telegram»

Claude Code будет редактировать файлы проекта прямо под твоим контролем.
