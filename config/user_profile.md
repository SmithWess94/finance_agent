# Профиль пользователя

## Кто я
Артём. Чайный мастер и проводник церемоний. Веду чайную, провожу чайные церемонии (в чайной, выездные, заказные) и саунд-хилинги.

## Доходы
- Нестабильные. Зависят от дня, сезона, количества церемоний.
- Источники: выручка из чайной, оплата церемоний, саунд-хилинги.
- В хорошие дни — несколько тысяч гривен, в тихие — почти ноль.

## Текущие проблемы
- Практически не веду учёт финансов — теряю понимание куда уходят деньги.
- Есть долги перед поставщиками чая.
- Нет финансовой подушки.

## Цели (приоритет по порядку)
1. Выработать привычку записывать все доходы и расходы каждый день.
2. Погасить долги перед поставщиками чая.
3. Накопить финансовую подушку — минимум 3 месяца расходов.
4. Выстроить понимание реальных цифр своего бизнеса.

## Горизонт
1–3 года до финансовой стабильности и свободы.

## Важное
- Советы должны учитывать нестабильность дохода — стратегии для фиксированной зарплаты не подходят.
- Ключевая задача сейчас — не оптимизация, а просто привычка фиксировать.
- Не нужна сложная теория — нужны конкретные шаги на сегодня.
