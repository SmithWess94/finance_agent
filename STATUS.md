## 🚀 Что делать дальше

### Немедленно (перед первым запуском)
1. Получи ключи (Anthropic, Telegram, Google)
2. Создай таблицу Google Sheets
3. Заполни `.env` файл
4. Запусти `python src/bot.py`

### После первого запуска
- Протестируй команды в Telegram
- Добавь первую запись о доходе/расходе
- Проверь, что данные сохраняются в Google Sheets

## 🔍 Инструкции по отладке

Если бот не работает, проверь:

```bash
# 1. Зависимости установлены?
pip list | grep anthropic

# 2. .env файл существует?
ls -la .env

# 3. Ключи правильные?
cat .env

# 4. Google credentials существуют?
ls -la config/google_credentials.json

# 5. Запусти бота с debug логами
python src/bot.py
```
