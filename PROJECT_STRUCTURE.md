## 🔧 Компоненты

### `bot.py` — Основной бот
- Класс `TeaAdvisor` — советчик на основе Claude API и знаний из книг
- Команды для добавления доходов, расходов, долгов
- Обработка естественного языка
- Ежедневные отчёты

**Команды:**
- `/start` — приветствие
- `/доход <сумма> <описание>` — добавить доход
- `/расход <сумма> <категория>` — добавить расход
- `/долг <поставщик> <сумма>` — добавить долг
- `/итоги` — отчёт за день
- Естественный язык: просто напиши сообщение

### `sheets_manager.py` — Google Sheets
- Создание листов (Доходы, Расходы, Долги, Прогресс)
- Добавление записей о доходах и расходах
- Управление долгами
- Получение отчётов

### `knowledge_base.py` — База знаний
- Загрузка всех .md файлов из папки knowledge_base/
- Интеграция с Claude API для советов

## 💾 Данные

### Google Sheets таблица содержит 4 листа:

1. **Доходы** — все источники дохода
   - Дата, Сумма, Источник, Категория

2. **Расходы** — все траты
   - Дата, Сумма, Категория, Комментарий

3. **Долги** — обязательства перед поставщиками
   - Поставщик, Изначальный долг, Погашено, Остаток

4. **Прогресс** — отслеживание целей
   - Дата, Размер подушки, Общий долг

## 🔐 Безопасность

- API ключи хранятся в `.env` (НЕ заливай в git)
- Google credentials в `config/google_credentials.json` (НЕ заливай в git)
- Файлы `.env` и `config/` добавлены в `.gitignore`
